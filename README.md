# It-Logger

React app to track IT department tasks and issues. With React hooks, Redux

**[Demo here!](https://pure-woodland-81783.herokuapp.com/)**

## Technologies

- React.js
- Redux
- JavaScript
- Materialize-CSS
- HTML
- Node.js
- Express.js
- MongoDB/mongoose

## Setup

```bash
git clone https://github.com/kulinar310/it-logger.git
cd it-logger
npm install
npm run dev
```
